# Lepus europaeus

This folder contains the scripts used to analyse ddRAD data from 10 individuals belonging to two populations of _Lepus europaeus_ sampled in Southern and Eastern Finland for the Mind the Gap project. 

## Author
Zsófia Fekete, University of Oulu, Finland
Jaakko Pohjoismäki, University of Eastern Finland, Finland
