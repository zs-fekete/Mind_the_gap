# Bombus terrestris

This folder contains the scripts used to analyse both RADseq and WGS data from 10 individuals belonging to three populations of _Bombus terrestris_. Individuals used in the RADSeq data were sampled in Central and Southern Portugal, whereas individuals used in the WGS were sampled in Central, South, and Northern Portugal.
